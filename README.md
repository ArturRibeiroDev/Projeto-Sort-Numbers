# 🔢 Sort Number

Aplicação web interativa que demonstra algoritmos de ordenação de números com interface visual e responsiva.

[![JavaScript](https://img.shields.io/badge/JavaScript-ES2020-f7df1e?style=for-the-badge&logo=javascript&logoColor=black)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
[![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)](https://developer.mozilla.org/en-US/docs/Glossary/HTML5)

---

## 🚀 Tecnologias

| Tecnologia | Função |
|---|---|
| **JavaScript** | Algoritmo de ordenação, DOM manipulation |
| **HTML5** | Interface |
| **CSS3** | Estilização responsiva com layout em grid |

## ✨ Funcionalidades

- ✅ Ordenação de números inseridos pelo usuário
- ✅ Interface responsiva com grid layout
- ✅ Feedback visual em tempo real

## 🚦 Como Rodar

Abra `index.html` no navegador. Não requer servidor.

## 💡 O que foi aprendido

- Algoritmos de ordenação implementados do zero
- Layout com CSS Grid e responsividade
- Interação entre input do usuário e processamento JS
